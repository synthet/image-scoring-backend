# Tech Stack

- **Language**: Python 3.10+
- **ML Framework**: TensorFlow (CPU), PyTorch (some models), Scikit-Learn.
- **Image Processing**: Pillow (PIL), rawpy, imageio.
- **Database**: Firebird SQL.
- **UI**: Gradio.
- **Server**: MCP (Model Context Protocol).
- **Testing**: Pytest.
- **Container**: Docker.
